# Roadmap

Harmonia is currently a local-first AI infrastructure stack, not yet a full platform.

## Direction

- **Now**: Reliable local stack for RAG, ML, data workflows, and LLM experimentation
- **Next**: Better DX, cleaner service boundaries, safer defaults, stronger RAG patterns, and stream capabilities
- **Later**: Security foundations, reusable service layers, and production-aligned deployment patterns
- **Future**: Evolve toward a fuller platform model with auth, policy, and stronger control-plane capabilities

## Priorities

- Keep the stack modular and easy to run locally
- Improve reproducibility and operational clarity
- Add stronger application-facing service abstractions
- Expand examples and reusable integration patterns

## Non-Goals Today

- Not a SaaS product
- Not a no-code tool
- Not yet a full AI platform
